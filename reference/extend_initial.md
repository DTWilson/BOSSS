# Extend the initial set of design evaluations

Extend the initial set of design evaluations

## Usage

``` r
extend_initial(problem, solution, extra_N = 0, extra_points = 0)
```

## Arguments

- problem:

  BOSSS problem.

- solution:

  current BOSSS solution.

- extra_N:

  number of extra simulations to run at exisiting designs.

- extra_points:

  number of new design points to add.

## Value

an updated BOSSS solution object
