Print the Pareto set of a BOSSS solution

## Usage

``` r
# S3 method for class 'BOSSS_solution'
print(x, ...)
```

## Arguments

- x:

  BOSSS solution.

- ...:

  No other arguments for this method.

## Value

A data.frame containing the Pareto set and the corresponding
